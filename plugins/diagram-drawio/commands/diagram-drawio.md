# draw\.io図作成コマンド

$ARGUMENTS

***

## 実行手順

1. **要件の理解**: ユーザーの説明から作成する図の種類と内容を理解する
2. **スキル参照**: `diagram-drawio` スキルのルールに従ってXMLを生成する
3. **出力**: .drawioファイルとして保存、またはターミナルに出力する

## 必須遵守事項

- フォント設定: すべてのテキスト要素に `fontFamily=Hiragino Sans` を明示的に設定
- 矢印配置: 矢印（edge）をXMLの先頭に配置し、ノード（vertex）を後に配置
- 日本語対応: 日本語テキストは1文字あたり30〜40pxの幅を確保
- フォントサイズ: 標準の1.5倍（18px程度）を使用

## 図の種類別ガイドライン

| 図の種類     | 主な要素              | 色の使い方                |
| -------- | ----------------- | -------------------- |
| フローチャート  | 開始/終了、処理、条件分岐     | 青（開始/終了）、緑（処理）、黄（条件） |
| シーケンス図   | アクター、ライフライン、メッセージ | 青（オブジェクト）、緑（DB）      |
| アーキテクチャ図 | コンポーネント、レイヤー、接続   | レイヤーごとに色分け           |
| ER図      | エンティティ、リレーション     | エンティティ種別で色分け         |

## 出力オプション

ユーザーの要件に応じて以下のいずれかで出力する。

1. ファイル保存: 指定されたパスに `.drawio` ファイルとして保存
2. ターミナル出力: XMLをそのまま出力（コピー＆ペースト用）
3. 両方: ファイル保存後、確認用にXMLも表示
